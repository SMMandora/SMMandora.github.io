Context

Most observability advice is slideware. This is a runnable stack that shows SLO practice end to end across a real three-service mesh — error budgets, burn-rate alerting, distributed tracing, and chaos, all as code.

Architecture & decisions

Three microservices (Go and Python) plus mock dependencies emit metrics, traces, and logs. Prometheus computes SLI, error budget, multi-window burn rate, and p95 as recording rules — one source of truth read by both a custom React SLO Control Center (8 tabs) and four Grafonnet dashboards. Alertmanager routes burn-rate, service-down, and disk alerts by severity to a webhook with linked runbooks. Trace context propagates through the Redis streams, so one payment is a single distributed trace.

Impact

Auto-verifying chaos scenarios (error burst, service down, payments-latency, disk fill) perturb the live stack, assert the expected alert fired, then revert. Runs on both Docker Compose and Kubernetes (kind + Kustomize), driven by deterministic k6 load, with the Prometheus recording rules unit-tested via promtool.

99.9% / 28d @ 200ms p95 · 3-service mesh · 4 dashboards · 8-tab UI · burn-rate alerting · chaos auto-verified

  • Context

    Code review is the bottleneck on small teams, and generic AI reviewers miss repo-specific conventions.

    Architecture & decisions

    GitHub webhook into an HMAC-validated FastAPI receiver. A LangGraph state machine fans out to four parallel check nodes, grounded by pgvector retrieval over past PR comments and style guides. Findings dedupe into one consolidated GitHub review.

    Impact

    Ships as one Docker container — FastAPI webhook, in-process worker, LangGraph agent, and a dashboard — with Postgres/pgvector its only dependency. Hits a p95 review under 30s on a 200-line diff at under $0.50 per review, gated by an eval suite of hand-labeled PRs (80%+ findings match) and configured per-repo via .codereview.yml.

  • Context

    Real-time fan-out at scale is the canonical distributed-systems problem: connections are stateful, everything else shouldn't be.

    Architecture & decisions

    Stateless Go gateways behind an L4 balancer with Redis pub/sub for cross-replica fan-out. A persistence worker batches writes to Postgres. Token-bucket rate limiting, presence via Redis sorted sets, graceful drain on shutdown.

    Impact

    Presence, typing, auth, history replay, and 30 msg/min token-bucket rate limiting — instrumented with Prometheus metrics and OpenTelemetry traces to Jaeger. Kubernetes manifests autoscale on connections-per-pod (HPA 2–20), and k6 ramps to 10k concurrent connections against a p99 fan-out target under 50ms.

  • Context

    Online exams need trustworthy proctoring that runs in real time and stays bandwidth-cheap — without streaming raw video to a server.

    Architecture & decisions

    The browser (React + Vite + TypeScript) runs MediaPipe Face Landmarker at ~15 FPS and emits deduplicated events plus occasional keyframes over WebSocket (~5–20 KB/s). A FastAPI backend runs YOLOv8 for objects and person-count, InsightFace ArcFace for identity rechecks, and Silero VAD with Whisper on audio.

    Impact

    Detects gaze, head pose, talking, multi-person, phones, tab-switching, and identity mismatch — driving a live proctor dashboard with auto-captured evidence clips and a per-session suspicion score, hardened with Fernet-encrypted ID storage and per-IP rate limiting.

devops practice

  1. 01

    Quality gates, not announcements

    Automated unit, integration, contract, and performance suites block regressions before they reach prod — they don't just notify after.

  2. 02

    Observability designed in, not bolted on

    SLI/SLO from day one. Grafana dashboards and proactive alerting paired with every service, not added after the first incident.

  3. 03

    Reproducible by default

    Docker and K8s declarative configs. CI is the only path to prod — local-only fixes don't ship.

  4. 04

    Test under load before users do

    Load and stress testing against production-like conditions catches bottlenecks before real traffic surfaces them.
